# Playwright Tictuk Test Automation

## Descripción
Automatización de flujo de compra end-to-end con verificación por API para Tictuk usando Playwright y TypeScript.

## Tecnologías
- Playwright
- TypeScript
- POM (Page Object Model)

## Requisitos
- Node.js v16+
- VS Code
- Windows 11
